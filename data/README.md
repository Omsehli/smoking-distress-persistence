# Data (ICPSR 36498)

This repository does NOT include PATH microdata files.

We use the Population Assessment of Tobacco and Health (PATH) Study
Public-Use Files (ICPSR 36498).

To reproduce the analysis:
1) Download the required PATH Public-Use files from ICPSR 36498 (you may need an ICPSR account).
2) Extract and place the raw files in:
   data/raw/

This project will generate cleaned/processed datasets in:
   data/processed/

Do NOT upload raw data to GitHub.
